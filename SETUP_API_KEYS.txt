===== AI ASSISTANT ZEN - API KEY SETUP GUIDE =====

Your assistant is now upgraded to be as smart as ChatGPT with Alexa-like voice!

IMPORTANT: You need to add API keys for the AI to work properly.

===== STEP 1: Create .env File =====
1. Create a new file named: .env (in this folder)
2. Add the following content to it:

===== STEP 2: Add API Keys to .env =====

# PRIMARY AI - ChatGPT (OpenAI) - RECOMMENDED
OPENAI_API_KEY=your_openai_api_key_here

# FALLBACK AI - Google Gemini (FREE)
GEMINI_API_KEY=your_gemini_api_key_here

# OPTIONAL - Wake Word Detection
PORCUPINE_ACCESS_KEY=your_porcupine_key_here
